# Word-Count## AIM:
To write a python program for getting the word count from a text.
## EQUIPEMENT'S REQUIRED: 
PC
Anaconda - Python 3.7
## ALGORITHM: 
### Step 1: Start the program.

### Step 2: Enter the text (sentence) from the user.
 
### Step 3: Store the entered text in a variable.

### Step 4:  Split the text into words using split() function.

### Step 5: Count the number of words using len() function.

### Step 6: Display the word count and stop the program.

## PROGRAM:
```
a=input()
b=a.split()
print(len(b))
```
